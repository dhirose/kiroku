文字列型のデータ操作

文字列の長さを取得

"hoge".length # 4

指定文字列が含まれるかチェックする

"hoge".contains("hoge") # True
"hoge".contains("fuga") # False

文字列がnullまたは空かチェックする

[string]::IsNullOrEmpty($Null) # True
[string]::IsNullOrEmpty("") # True
[string]::IsNullOrEmpty(" ") # False
[string]::IsNullOrEmpty("hoge") # False

文字列を置換する

"abc,abc".Replace("a", "A") # Abc,Abc

文字列の開始位置を取得する

"abc,abc".IndexOf("b") # 1

文字列の終了位置を取得する

"abc,abc".LastIndexOf("b") # 5

パディング文字を左に埋める

"hoge".PadLeft(10, "A") # AAAAAAhoge ※第1引数に達するまで、第2引数でパディングする
"hoge".PadLeft(10)      #       hoge ※第2引数がない場合は空白でパディングをする

パディング文字を右に埋める

"hoge".PadRight(10, "A") # hogeAAAAAA ※第1引数に達するまで、第2引数でパディングする
"hoge".PadRight(10)      # hoge       ※第2引数がない場合は空白でパディングをする

文字列を指定した文字列で分割する

"abc,abc,abc".Split(",") # カンマで分割
"abc abc abc".Split("")  # 空白(" ")で分割
"abc abc abc".Split()    # 空白(" ")で分割

<# 上記実行結果はすべて同じ
abc
abc
abc
#>

指定位置から、指定文字数の文字列を取得する

"hoge".Substring(1,2) # og

文字列を1文字ごとに区切って配列にする

"hoge".ToCharArray()
<#
h
o
g
e
#>

小文字を大文字に変換する

"hoge".ToUpper() # HOGE

大文字を小文字に変換する

"HOGE".ToLower() # hoge

文字列の両端から空白を削除する

"   hoge   ".Trim() # hoge

文字列の左端から空白を削除する

"   hoge   ".TrimStart() # hoge

文字列の右端から空白を削除する

"   hoge   ".TrimEnd() #    hoge
