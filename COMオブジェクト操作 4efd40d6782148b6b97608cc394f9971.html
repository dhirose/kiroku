      • [void][System.Runtime.InteropServices.Marshal]::FinalReleaseComObject($xl)
        • COMオブジェクトの参照の解放
        • 先頭のvoidは戻り値を持たないコマンドにするためのもの。指定しなかった場合、戻り値が画面に出力される。0の場合、コマンドが成功。
        • このコマンドを実行しなかった場合、コンピューター上にこのスクリプトを実行した残骸(使用されなくなったプロセス)が残る。タスクマネージャーから確認できる。（プロセスの「EXCEL.EXEがそれ」）

# Excelアプリケーションのインスタンスを生成する
$xl = New-Object -Com Excel.Application

# Excelアプリケーションを表示する

$wb = $xl.Workbooks.Open($xlfile)

# Sheet1を選択する

# Excelファイルを閉じる
$wb.Close()

$xl.Quit()

# COMオブジェクトの参照を解放する
[void][System.Runtime.InteropServices.Marshal]::FinalReleaseComObject($xl)
    • Excelファイルを開く場合
      • 「$xl.Visible = $true」を設定する
        • Excelファイルを開いてスクリプトが実行されるため、処理時間が延びる
    • Excelファイルを開かない(バックグランド実行)場合
      • 「$xl.Visible = $true」を設定しない。または、$xl.Visible = $falseを設定する

# COMオブジェクトの参照を解放する
[void][System.Runtime.InteropServices.Marshal]::FinalReleaseComObject($xl)
    • 数式を入れる場合
      • 「$ws.Cells.Item(2,1) = “=NOW()"」のように記載する（現在日時を入れる）
    • 新たなExcelファイルとして保存する場合
      • 「$wb.Save()」を「$wb.SaveAs([ref]”C:\Users\hirok\!work\test2.xlsx”)」のようにすると新たなExcelファイルで保存できる。
      • SaveAsメソッドは「名前を付けて保存」に該当する。そのため、すでにファイルが存在する場合、上書きのメッセージが表示される。上書きのメッセージを表示させないようにするには「$xl.DisplayAlerts = $false」を設定する。PawerShellスクリプト上から一切操作によるメッセージを表示しないようにしたい場合は、COMオブジェクトを参照した直後、すぐにDisplayAlertsプロパティを$falseに設定するとよい。
        $xlfile = "C:\Users\hirok\!work\test.xlsx"

# COMオブジェクトの参照を解放する
[void][System.Runtime.InteropServices.Marshal]::FinalReleaseComObject($xl)
    • ファイル形式を変換して保存する
      • SaveAsメソッドのパラメータにファイル形式を指定する
        • 参考コード：$wb.SaveAs([ref]”C:\Users\hirok\!work\test2.csv”,6) # 6はCSV形式
        • 設定できる値は、XlFileFormat列挙体に定義されている
          • http://msdn.microsoft.com/ja-jp/library/office/ff198017(v=office.15).aspx
    • Excelファイルを新規作成する
      • WorkbooksクラスのAddメソッドを実行して、Excelファイルを新規作成する
        • $wb = $xl.Workbooks.Add()
      • 参考スクリプト
        # Excelアプリケーションのインスタンスを生成する
$xl = New-Object -Com Excel.Application

# Excelファイルを作成し、そのオブジェクトを変数$wbに格納する
$wb = $xl.Workbooks.Add()

# Sheet1を選択する
$ws = $wb.Worksheets.Item("Sheet1")

# A1セルの内容を更新する
$ws.Cells.Item(1,1) = "Hello World!"

# Excelファイルを新たなファイルとして保存する
$wb.SaveAs([ref]”C:\Users\hirok\!work\test2.xlsx")

# Excelファイルを閉じる
$wb.Close()

# Excelアプリケーションを終了する
$xl.Quit()
