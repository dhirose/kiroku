レジストリの操作

  • レジストリを操作する前に、レジストリのバックアップを取る
  • コマンドから操作できるレジストリは以下の2つ
    • HKEY_LOCAL_MACHINE
    • HKEY_CURRENT_USER
  • レジストリの切り替え
    • 構文：Set-Location <ドライブ名>:
    • 実行例
      • HKLM：HKEY_LOCAL_MACHINE
      • HKCU：HKEY_CURRENT_USER
      PS C:\Users\hirok\!work\powerShell> Set-Location HKLM:
PS HKLM:\> Set-Location HKCU:
PS HKCU:\>
    • 「Convert-Path <ドライブ名>:」で正式なレジストリキーが参照できる。
      PS HKCU:\> Convert-Path HKCU:
HKEY_CURRENT_USER\
  • レジストリキーの存在チェック
    • 構文：Test-Path <レジストリキー>
    • 実行例
      • Softwareを検査する場合
        PS C:\Users\hirok\!work\powerShell> Test-Path "HKCU:\Software"
True
PS C:\Users\hirok\!work\powerShell>
  • レジストリキーを取得する
    • 構文：Get-Item [レジストリキー]
    • 実行例
      PS C:\Users\hirok\!work\powerShell> Get-Item HKCU:


    Hive:


Name                           Property
----                           --------
HKEY_CURRENT_USER


PS C:\Users\hirok\!work\powerShell>
      • [レジストリキー]を指定しない場合は、現在のレジストリを取得する

  • レジストリキー追加する
    • 構文：New-ItemProperty [レジストリキー] [エントリ名] [エントリの種類] [レジストリ値]
    • ※レジストリ操作はあまりすることがないため、構文だけメモして、時間があれば勉強する。
  • レジストリキーやエントリ、値を変更する
    • 構文：Rename-Item [変更前のレジストリキー] [変更後のレジストリキー]
    • 構文：Rename-ItemProperty [レジストリキー] [変更前のエントリ名] [変更後のエントリ名]
    • 構文：Set-ItemProperty [レジストリキー] [エントリ名] [レジストリ値]
    • ※レジストリ操作はあまりすることがないため、構文だけメモして、時間があれば勉強する。
  • レジストリのエントリを削除する
    • 構文：Remove-ItemProperty [レジストリキー] [削除するエントリ名]
    • ※レジストリ操作はあまりすることがないため、構文だけメモして、時間があれば勉強する。
  • レジストリを削除する
    • 構文：Remove-Item [レジストリキー]
    • ※レジストリ操作はあまりすることがないため、構文だけメモして、時間があれば勉強する。
